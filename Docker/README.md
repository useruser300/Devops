# Docker Fundamentals & Reference

This folder contains a growing Docker knowledge base focused on
understanding core concepts before moving to advanced and production-level usage.

The documentation is built step by step, starting from fundamentals
and expanding over time according to a clear learning roadmap.

This repository reflects an active learning process, not a finished product.

---

## Current Focus

The current focus is on understanding Docker fundamentals:

- What Docker is and why it exists
- How containers work
- The difference between images and containers
- How images are stored and shared
- How Docker Compose fits into multi-container setups

These concepts are the foundation for everything else in Docker.
